## Features

- View all launchd agents and daemons (User/System)
- Enable/Disable services with one click
- Search and filter services
- View service configurations
- Open logs in Console.app
- Reveal plist files in Finder

## Requirements

- macOS 14.0+
- Apple Silicon (arm64)

## Installation

1. Download `LaunchWarden.dmg` from Releases
2. Drag `LaunchWarden.app` to Applications
3. Run and enjoy!
